Application Description

SyllaBye is an application created to help professors and other university administrators with the process of creating and updating the syllabi for different classes and departments. Class syllabi tend to change every semester with new requirements, such as: updated classroom content, health and safety guidelines, links to campus resources, schedule changes, and other information. The end goal of this software/program is to assist the user by making this simple process less time-consuming to allow educators to focus and spend more time devoted to teaching and helping students.

Basic Features

  • Simple and intuitive UI.
  • Site menu for easy navigation.
  • Home page with introduction text and link to this (proposal) page.
  • Contact/team page.
  • Suggestion-form page.
  • Server pinging (for testing purposes).

Syllabus Creation Features

  • Convert syllabi to a printable PDF.
  • Generate/download syllabi as a printable PDF.
  • (beta) Consistent formatting.
  • (beta) Custom inputs automatically formatted upon generation.
  • (beta) Customization options like changing the font or banner.
  • (beta) Semester Calendar.

Upcoming (Future) Features
